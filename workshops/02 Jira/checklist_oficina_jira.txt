Checklist de atividades

	- Criar alguns itens no Product Backlog de vários tipos (histórias, tarefas, bugs e análises)
	- Adicionar descrição na parte interna dos itens para fornecer mais detalhes
	- Renomear itens para ter títulos melhores seguindo alguns padrões de histórias ágeis.
	- Ordenar itens no Backlog
	- Excluir itens desnecessários do Backlog
	 
	- Adicionar itens no Sprint Backlog
	- Atribuir um Responsável para cada item da Sprint
	- Iniciar a Sprint
	- Mudar status dos itens para Em Andamento
	- Adicionar comentários nos itens para descrever o trabalho realizado
	- Finalizar um item concluído
	- Finalizar a Sprint!

Adicionar descrição nos itens:
	A Livro - Permitir cadastro
		Criar uma tela para cadastrar um novo livro. Os campos obrigatórios são "Título, Autor, ISBN", os campos opcionais são "Ano, Editora, e número de páginas"
	
	B Mudar a cor da barra superior
		Alterar a cor da topbar para a mesma cor que está atualmente da sidebar (azul claro). Utilizar mesmo código da cor.
	
	C Relatórios - Permitir gerar um Relatório de Empréstimos por data inicial/final
		No menu Relatórios, criar um sub-menu que abre uma tela que permite ao usuário inserir as datas, e um botão para gerar Relatório em PDF. Colunas do relatório "Data, Livro, Usuário".
	
	D Livro - Permitir editar 
		Criar uma tela para editar os campos de um livro.
	
	E Gênero - Permitir cadastrar um novo gênero de livro
		Criar uma tela para cadastrar um novo gênero de livro. Campo obrigatório: Descrição.
	
	F Exibir "Bom dia" após o login
		Ao efetuar o login, um popup deve aparecer com a mensagem de BomDia/BoaTarde/BoaNoite e um versículo aleatorio extraido do arquivo csv em anexo.
	
	G Livro - Permitir excluir
		Permitir excluir (soft delete) um livro.   O mecanismo de recuperação do livro será implementador em uma outra história. 
	
	H Livro - Permitir buscar pelo título
		Na tela principal de Livros, a que exibe todos os livros, criar um campo de busca pelo título.   Este campo deve ignorar espaços em brancos e permitir buscar livros por parte do nome (ilike).
